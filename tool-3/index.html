Get Started

Getting Started

Ensure a smooth experience by following these guidelines:

  • Ensure you are connected to the internet.
  • Verify app permissions and enable them fully.
  • Dark mode is supported, but not fully tested.
  • Ensure your phone language is set to English (not Arabic).
  • If WhatsApp crashes, adjust the message delay between 5-10 seconds.
  • Important: Group extractor is known to crash the app—do not use it.
  • WhatsApp Business is supported but not fully tested.

Troubleshooting

App Permissions Issue

Ensure that all permissions are enabled for the app to function properly. Go to Settings > Apps > Permissions.

WhatsApp Slows Down

If WhatsApp is slow or crashes, try adjusting the message delay in the app settings. A 5000-10000 ms delay works best.

Group Extractor Crashes

Due to ongoing development, the group extractor is not functional and will cause the app to crash. Avoid using it for now.
